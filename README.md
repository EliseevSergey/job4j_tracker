# job4j_tracker
Консольное приложение(Package tracker). Пользователю отображается меню с возможностями программы.
Программа может:
1. Добавлять заявку в базу данных pSql
2. Заменять заявку на новую заявку по ID.
3. Удалять заявку по ID.
4. Отображать список всех заявок.
5. Производить поиск по имени заявки.
______________
Кроме проекта Tracker в репозитории расположены решения заданий курса job4j по коллекциям, лямбда и тд.
